Field reference for AirData API data

The fields below are returned in the Sample API.

Capacity Fields
Field name Description Data type
data_provider Data provider placeholder string
departure_airport IATA Departure Airport string
departure_country Departure Country Code string
airline Airline Code string
arrival_airport IATA Arrival Airport string
arrival_country Arrival Country Code string
aircraft Aircraft IATA string
flight_number Carrier's Flight Number string
dep_day Departure Day (Monday,Tuesday, ...) string
seats Total seats for this flight int
is_lowcost Lowcost flag bool
dep_date Departure date datetime
arr_date Arrival Date datetime
